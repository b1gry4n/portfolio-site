# Website Portfolio

This folder is a fully contained portfolio version of the doctor-note extraction project.

## What it includes

- a professional case-study style website
- a working in-browser doctor-note extraction demo
- a visible clarification-dialogue section
- an evaluation section with expected vs actual outputs

## Privacy

Use only fake or anonymized notes in this website. Do not use real medical notes.

## Run locally

Open `index.html` directly in a browser.

Optional local server:

```powershell
cd website_portfolio
& "C:\Users\ryans\.cache\codex-runtimes\codex-primary-runtime\dependencies\python\python.exe" -m http.server 8000
```

Then open:

`http://localhost:8000`

## Self-contained

This folder does not depend on files outside `website_portfolio` for the website demo to work.
